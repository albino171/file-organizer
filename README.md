# 📂 File Organizer

Projeto em Python que organiza automaticamente arquivos em pastas por tipo.

## 🚀 Funcionalidades
- Organiza arquivos automaticamente
- Separa por tipo (imagens, documentos, vídeos, etc.)
- Cria pastas automaticamente se não existirem

## 🧠 Tecnologias
- Python 3

## ▶️ Como usar
Coloque o script na pasta que deseja organizar e execute:

```bash
python organizer.py
